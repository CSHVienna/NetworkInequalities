Node vectors¶

class netin.graphs.NodeVector(n: int, dtype: Type, fill_value: Number | None = None, name: str | None = None)¶

Wrapper for numpy arrays that represents node values. The number of nodes does not necessarily correspond to the number of nodes in the graph. It implements an interface with numpy to allow the usage of numpy functions.

Parameters:
nint

The number of nodes. Does not necessarily correspond to the number of nodes in the graph.

dtypeType

The data type of the values, must be a valid numpy data type.

fill_valueOptional[Number], optional

If the NodeVector should be initialized with an arbitrary value, by default None

nameOptional[str], optional

Name of the NodeVector, by default None

class netin.graphs.CategoricalNodeVector(n: int, n_values: int, class_labels: List[str] | None = None, fill_value: Number | None = None, name: str | None = None)¶

Initializes a CategoricalNodeVector. This class represents a vector of categorical node values. For instance, it can be used to represent the class labels of nodes.

Parameters:
nint

Number of nodes.

n_valuesint

Number of possible values.

class_labelsOptional[List[str]], optional

Labels for the classes. If provided, its size must match n_values, by default None

fill_valueOptional[Number], optional

The filling value, by default None

nameOptional[str], optional

The vector’s name, by default None
