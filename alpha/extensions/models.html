Model¶

class netin.models.Model(*args, N: int, seed: int | Generator | None = None, **kwargs)¶

Abstract modelling class. This class defines a growing network model. Specific growing-network-model implementations should inherit from this class and implement the provided abstract methods.

EVENTS: List[Event]¶

Evokes Event.SIMULATION_START and Event.SIMULATION_END.

SHORT = 'MODEL'¶

A shorthand for the model name

abstract _initialize_empty_graph() → Graph¶

Initializes an empty graph. This should be overwritten by the actual model implementations. See Model._simulate() for the initialization order.

_initialize_filters()¶

Initializes the filters. Default filters are no self loops and no double links.

_initialize_graph()¶

Initializes the empty graph, populates it and set the number of total nodes.

abstract _initialize_lfms()¶

Initialize the link formation mechanisms. This should be overwritten by the actual model implementations. See _simulate() for the initialization order.

Initializes the node classes. This should be overwritten by the actual model implementations. See _simulate() for the initialization order.

initialize_simulation()¶

Initializes the simulation of the model. The order of initialization is

  1. (Optional) Graph initialization if not preloaded.

  2. Node classes initialization.

  3. Filters initialization.

  4. Link formation mechanisms initialization.

simulate() → Graph¶

Runs the simulation. This calls Model.initialize_simulation() and Model._simulate(). Check these functions in case you want to extend Model. Triggers the Event.SIMULATION_START and Event.SIMULATION_END events.

Returns:
Graph

The simulated graph.

Notes

Triggers the following events.

  • Event.SIMULATION_START: When the simulation starts.

  • Event.SIMULATION_END: When the simulation ends.

_initialize_node_classes()¶

Initializes the node classes. If the attributes are present already (for instance, for pre-loaded graphs), they are kept and extended to the total number of nodes. Otherwise, a new binary class node vector is created.

class netin.models.UndirectedModel(*args, N: int, m: int, seed: int | Generator | None = None, **kwargs)¶

The base class for all undirected models. Based on [BarabasiAlbert1999], this model grows a network by adding a total of N nodes to the network. Each node that is added connects to the previously added nodes with m links. How the target nodes are chosen depends on link formation mechanisms. The implementation of these mechanisms is handled by subclasses (see for instance, PAHModel).

Parameters:
Nint

Number of nodes to be added.

mint

Number of links for each new node.

seedUnion[int, np.random.Generator], optional

The randomization seed or random number generator, by default 1

Populates the initial graph. This should be overwritten by the actual model implementations. See Model._simulate() for the initialization order.

Returns:

The populated graph.

_simulate() → Graph¶

Simulates the undirected model. After adding m initial nodes, N-m nodes are added one after the other. Each new node connects to previously added nodes with m links. The choice of target nodes depends on the implementation of LinkFormationMechanism and Filter. This should be implemented in respective subclasses.

Returns:
Graph

The simulated network.

class netin.models.DirectedModel(*args, N: int, f_m: float, d: float, plo_M: float, plo_m: float, seed: int | Generator | None = None, **kwargs)¶

The DirectedModel defines a network growth model which selects source nodes based on their activity and parameterized by power law exponents plo_m/M. Edges are created until a total network density d is reached. Subclasses should implement the way in which target nodes are chosen. See [Espin-Noboa2022] for details.

Parameters:
Nint

The number of nodes to be added.

f_mfloat

The fraction of minority nodes.

dfloat

Edge density to be reached.

plo_Mfloat

Power law exponent for the majority activity.

plo_mfloat

Power law exponent for the minority activity.

seedUnion[int, np.random.Generator], optional

Randomization seed or random number generator, by default 1
