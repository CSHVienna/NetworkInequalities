Event Handling¶

Several other classes implement the HasEvents interface. In that case, they should provide a HasEvents.EVENTS list that contains the Events that they implement.

If you wish to inject your own code, check HasEvents.register_event_handler(). As an example, consider the (shortened) implementation of SimulationTimer:

class SimulationTimer:
    _start_time: float # Starting time

    time: Optional[float] = None # Final measure

    def __init__(self, model: Model) -> None:
        # Start timer with simulation
        model.register_event_handler(
            Event.SIMULATION_START, self._start_timer)

        # End timer with simulation
        model.register_event_handler(
            Event.SIMULATION_END, self._end_timer)

    def _start_timer(self):
        # Sets the starting time
        self._start_time = time.time()

    def _end_timer(self):
        # Computes the final simulation run
        self.time = time.time() - self._start_time

This code registers two functions to measure the simulation. When the simulation starts, the _start_timer() method is called to store the starting time. Once the simulation ends, _end_timer() takes again the time and subtracts the starting time. After the simulation, the time attribute contains the runtime of the simulation.

class netin.utils.Event(value)¶

Specifies events that are triggered when the specified action occurs.

LINK_ADD_AFTER¶

After an edge is added to a graph.

LINK_ADD_BEFORE¶

Before an edge is added to a graph.

SIMULATION_END¶

When the model simulation terminates.

SIMULATION_START¶

When the model simulation starts.

TARGET_SELECTION_GLOBAL¶

When a target node in PATCHModel is chosen globally.

TARGET_SELECTION_LOCAL¶

When a target node in PATCHModel is chosen locally.

class netin.utils.HasEvents(*args, **kwargs)¶

Interface to be implemented when a class triggers events.

EVENTS: List[Event]¶

List of implemented events. Should be overwritten by classes that use this interface.

register_event_handler(event: Event, function: Callable[[Any], None])¶

Registers a new callback function for a given event. This can be used to inject code which will be executed when the respective event is called.

Parameters:
eventEvent

The event to register the callback for.

functionCallable[[Any], None]

The function to be called when the event is triggered.

remove_event_handler(event: Event, function: Callable[[Any], None] | None = None)¶

De-registers an event handler.

If function is provided, only the specific function is removed. Otherwise, all functions are deleted.

Parameters:
eventEvent

The Event to remove the handling function from.

functionOptional[Callable[[Any], None]], optional

The function to be removed, by default None. If None, all functions are removed. Otherwise, only the specified ones will be deleted.

trigger_event(*args, event: Event, **kwargs)¶

Triggers an event and calls all registered callback functions. This function should be called by the class implementing classes.

Parameters:
eventEvent

The event to be triggered.
