Directed Graph Models¶

class netin.models.DHModel(*args, n: int, f_m: float, d: float, plo_M: float, plo_m: float, h_mm: float, h_MM: float, seed: int | Generator | None = None, **kwargs)¶

The DHmodel is a directed model that joins new nodes to the existing nodes with a probability proportional to the group assignment of the source and target nodes.

Parameters:
nint

Number of nodes to add.

f_mfloat

The minority group fraction.

dfloat

The target network density to reach.

plo_Mfloat

The power law exponent for the majority activity.

plo_mfloat

The power law exponent for the minority activity.

h_mmfloat

The homophily of the minority nodes.

h_MMfloat

The homophily of the majority nodes.

seedUnion[int, np.random.Generator, None], optional

Randomization seed or generator, by default None. If None, each run will be different.

EVENTS: List[Event]¶

Evokes Event.SIMULATION_START and Event.SIMULATION_END.

SHORT = 'DH'¶

A shorthand for the model name

compute_target_probabilities(source: int) → ndarray¶

Computes the target probabilities based on TwoClassHomophily.

Parameters:
sourceint

The source node.

Returns:
np.ndarray

The target node probabilities.

get_metadata(d_meta_data: Dict[str, Any] | None = None) → Dict[str, Any]¶

Returns the metadata of the model.

Returns:
Dict[str, Any]

Metadata of the model. It includes the density d, the activity power law exponents plo_M and plo_m.

initialize_simulation()¶

Initializes the simulation of the model. The order of initialization is

  1. (Optional) Graph initialization if not preloaded.

  2. Node classes initialization.

  3. Filters initialization.

  4. Link formation mechanisms initialization.

preload_graph(_: DiGraph)¶

Preloads a graph into the model. This is currently not supported for the directed models.

Parameters:
_DiGraph

The graph to preload.

register_event_handler(event: Event, function: Callable[[Any], None])¶

functionCallable[[Any], None]

remove_event_handler(event: Event, function: Callable[[Any], None] | None = None)¶

De-registers an event handler.

If function is provided, only the specific function is removed. Otherwise, all functions are deleted.

Parameters:
eventEvent

The Event to remove the handling function from.

functionOptional[Callable[[Any], None]], optional

The function to be removed, by default None. If None, all functions are removed. Otherwise, only the specified ones will be deleted.

simulate() → Graph¶

Runs the simulation. This calls Model.initialize_simulation() and Model._simulate(). Check these functions in case you want to extend Model. Triggers the Event.SIMULATION_START and Event.SIMULATION_END events.

Returns:

The simulated graph.

Notes

Triggers the following events.

  • Event.SIMULATION_START: When the simulation starts.

  • Event.SIMULATION_END: When the simulation ends.

class netin.models.DPAModel(*args, n: int, f_m: float, d: float, plo_M: float, plo_m: float, seed: int | Generator | None = None, **kwargs)¶

The DPAModel is a directed model that joins new nodes to the existing nodes with a probability proportional to the in-degree of the existing nodes.

For the simulation logic of this model, see the base DirectedModel.

class netin.models.DPAHModel(*args, n: int, f_m: float, d: float, plo_M: float, plo_m: float, h_mm: float, h_MM: float, seed: int | Generator | None = None, **kwargs)¶

The DPAHModel is a directed model that joins new nodes to the existing nodes with a probability proportional to the in-degree of the existing nodes and the homophily of the nodes. See [Espin-Noboa2022] for details.
